# 5.6.4. Dọn dẹp Amazon API Gateway

1. Truy cập dịch vụ **Amazon API Gateway** -> chọn **APIs**.
2. Chọn API đã khởi tạo (`FightingGameAPI`), chọn **Delete** và nhập xác nhận để xóa hoàn toàn API Gateway.
